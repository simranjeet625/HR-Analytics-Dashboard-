# HR Analytics Dashboard

## Project Description
HR Analytics Dashboard project built using Tableau to analyze employee data and visualize key HR metrics such as attrition rate, salary distribution, job roles, and department-wise performance. The project helps in understanding workforce trends and supports data-driven HR decision-making.

## Tools Used
1. Tableau  
2. Excel / CSV Dataset  

## How I Built This Project
1. Collected and understood HR dataset  
2. Performed data cleaning and preparation  
3. Conducted Exploratory Data Analysis (EDA) to find patterns  
4. Designed interactive dashboards in Tableau  
5. Used charts, KPIs, and filters for visualization  
6. Generated insights from the analysis  

## Features
1. Employee attrition analysis  
2. Salary distribution analysis  
3. Department-wise workforce analysis  
4. Job role analysis  
5. Interactive filters and dashboards  

## Key Insights
1. Identified major factors affecting employee attrition  
2. Analyzed salary impact on employee retention  
3. Studied department-wise employee distribution  
4. Highlighted high-risk employee groups  

## Learning Outcomes
1. Learned data analysis and visualization using Tableau  
2. Improved understanding of HR analytics concepts  
3. Gained experience in data cleaning and EDA  
4. Developed skills to build interactive dashboards  
5. Learned to convert raw data into meaningful insights  

## Conclusion
This project demonstrates how HR data can be analyzed using Tableau to generate meaningful insights that help in better decision-making and understanding employee behavior.
